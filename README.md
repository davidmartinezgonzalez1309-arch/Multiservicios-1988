# Multiservicios y Reformas 1988

Página web de reformas integrales en Albacete, Valencia, Alicante, Madrid y Murcia.

## Servicios

- Albañilería
- Fontanería completa
- Electricidad completa
- Trabajo de pladur
- Pintura y decoración
- Trabajos en altura y reparación de tejados

## Cómo usar

1. Abre `index.html` en tu navegador
2. Edita el bloque `CONFIG` al inicio del `<script>` para cambiar datos del negocio
3. Sustituye las imágenes por las tuyas en la misma carpeta
